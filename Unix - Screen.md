## Commands

```
screen -r              # reattach
screen -L              # automatically log to screenlog.n
```

## Keyboard sequences

```
C-a n     next
C-a tab
C-a b     break
C-a d     detach
C-a k y   kill
C-a C-a   send C-a
C-a C-\   terminate
C-a [     scrollback mode
C-a |     vertical split
C-a c     new screen and shell in blank pane
C-a H     toggle logging
C-a :     change screen working directory for logging

```
